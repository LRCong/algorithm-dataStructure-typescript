## 第一章 通用排序
#### 基础的排序算法：插入排序和选择排序
选择排序的特点是无论数组是否部分有序，比较次数都不变，而插入排序可以有效应对部分有序数组
#### 希尔排序
为了优化插入排序对大规模无序数组的排序效率，引入希尔排序
#### 两种经典NlogN排序
归并排序至多6NlogN次比较，快速排序是适用性很强的排序算法
#### 堆排序
堆（优先队列）这种数据结构可以有效应对一些场景，比如任务调度，基于堆的堆排序是一种优雅的排序算法。
